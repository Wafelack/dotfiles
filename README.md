# dotfiles

My dotfiles for GNU/Linux, managed with [rdfm](https://github.com/wafelack/rdfm).

## Software

- Window Manager   : [i3](i3/)
- Status bar       : [i3blocks](i3blocks/)
- Dmenu            : [rofi](rofi/)

---

- Terminal emulator: [kitty](kitty/)
- Shell            : [fish](fish/)
- Code Editor      : [Visual Code](VisualCode/)
- Text Editor      : [vim](vim/)
